Required Excel Columns

Your Excel must include the following columns:

  • SNo
  • Firstname
  • Unique ID (Employee ID / Email)
  • Usergroup
  • Reporting Manager 1
  • Reporting Manager 2
Processing Logic

The system will:

  1. Read users and their reporting managers.
  2. Create a unique Manager list from Manager 1 & Manager 2.
  3. Identify Direct Managers (listed in M1/M2).
  4. Identify Indirect Managers (higher-level reporting chain).
  5. Generate a new sheet named Manager Summary with Manager / Usergroups / Type.
  6. Return processed Excel for download (no storage).
